# Walmart Sales Executive Summary (Google Sheets)

## Project Overview
This project presents an executive-level sales summary created using Google Sheets. The objective was to organize and summarize sales data in a clear and structured format to support administrative, operational, and business decision-making.

## Tools Used
- Google Sheets
- Spreadsheet formulas and functions

## Analysis Process
- Reviewed and organized sales data
- Created summary tables and key performance metrics
- Analyzed sales trends and performance indicators
- Structured the file for executive-level reporting

## Key Insights
- Clear visibility into sales performance
- Identification of trends and patterns across sales data
- Well-organized summary suitable for executive reporting

## File Included
- `walmart_sales_executive_summary.xlsx`: Sales summary and executive report

## Notes
This project focuses on practical reporting and business analysis skills. The analysis was performed entirely in Google Sheets and exported for portfolio documentation.
